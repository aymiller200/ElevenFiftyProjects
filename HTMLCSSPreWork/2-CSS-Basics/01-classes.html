Classes

A CSS class in an identifier of an element which can be used muliple times on a page.

Memory device: Classes in school are periods. Remember to use a . whenever you want to refernce a class.

  • Classes start with . in the selector .anything-you-want-to-name-it
  • the same class name can be used more than once within the smae file
  • The class keyword refers to the class name that you set in the html page in the specific tag.
  • In the CSS file, you can identify a class on the whole, or you can identify particular parts of style (ex: p tags, h3, or li tags)
